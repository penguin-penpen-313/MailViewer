Google OAuth2 + Gmail API の設定手順
⏱ 所要時間: 約10〜15分
☁️
Google Cloud
無料アカウントでOK
🔑
Client ID取得
5ステップで完了
🚀
Netlifyで公開
ドラッグだけ
手順
1
Google Cloud Console を開く
▾
1
ブラウザで以下のURLを開く
https://console.cloud.google.com/
Googleアカウントでログインしてください。アカウントがない場合は無料で作成できます。
2
新しいプロジェクトを作成
Google Cloud Console
📁 プロジェクトの選択 ▾
↓ 「新しいプロジェクト」をクリック
＋ 新しいプロジェクト →
プロジェクト名は MailFocus など何でもOKです。「作成」をクリック。
2
Gmail API を有効にする
▾
1
左メニューから「APIとサービス」を選択
左側メニュー
🏠 ホーム
🔌 APIとサービス ▸
📚 ライブラリ
2
「ライブラリ」→ 「Gmail API」を検索
検索欄に Gmail と入力すると「Gmail API」が表示されます。クリックして「有効にする」ボタンを押してください。
✅
「APIが有効です」と表示されれば完了です
3
OAuth 同意画面を設定する
重要
▾
ℹ️
この設定をしないとClient IDが作れません。手順通りに進めてください。
1
「APIとサービス」→「OAuth同意画面」を開く
APIとサービス
🔑 認証情報
📋 OAuth同意画面 ←ここ
📚 ライブラリ
2
ユーザーの種類: 「外部」を選択して「作成」
個人利用の場合は「外部」でOKです。
3
アプリ情報を入力
アプリ名MailFocus
サポートメール自分のGmailアドレス
デベロッパー連絡先自分のGmailアドレス
入力したら「保存して次へ」をクリック。
4
スコープはそのまま「保存して次へ」
スコープの追加は不要です。そのまま次に進んでください。
5
テストユーザーに自分のメールアドレスを追加
「＋ ADD USERS」をクリックして、自分のGmailアドレスを入力。「保存して次へ」で完了。
⚠️
テストユーザーに追加したアドレスのみログインできます。必ず自分のアドレスを追加してください。
4
OAuth クライアント ID を作成する
本題
▾
1
「APIとサービス」→「認証情報」を開く
APIとサービス
🔑 認証情報 ←ここ
📋 OAuth同意画面
2
「＋ 認証情報を作成」→「OAuthクライアントID」
認証情報ページ上部
➕ 認証情報を作成 ▾
🔑 OAuthクライアントID ←クリック
🔐 サービスアカウント
3
アプリの種類: 「ウェブアプリケーション」を選択
ドロップダウンから「ウェブアプリケーション」を選んでください。
4
承認済みのJavaScript生成元にURLを追加
「承認済みの JavaScript 生成元」の「＋ URIを追加」をクリックして、アプリを公開するURLを入力します。
入力するURL（どちらか使う方）
Netlifyの場合:
https://あなたのサイト名.netlify.app
GitHub Pagesの場合:
https://あなたのID.github.io
ローカルでテストする場合も追加:
http://localhost:8080
⚠️
URLは先にNetlifyかGitHub Pagesで公開してから入力するのがベストです。後から追加・変更できます。
5
「作成」→ Client IDをコピー
ダイアログに表示される クライアントID（xxxxxxxxxx.apps.googleusercontent.com という形式）をコピーしてください。
🎉
これでClient IDの取得完了です！
5
Netlifyにアップロードして公開する
▾
1
4ファイルを同じフォルダに用意する
📁 MailFocus/
├── index.html
├── manifest.json
├── sw.js
└── README.md
2
Netlify を開く
https://netlify.com
「Sign up」から無料アカウントを作成（GitHubアカウントでもログイン可）。
3
フォルダをドラッグ＆ドロップ
📁
MailFocusフォルダをここにドロップ
Sites ページの下部にドロップゾーンがあります
数秒で公開完了！ https://ランダム文字.netlify.app というURLが発行されます。
4
発行されたURLをGoogle Cloudに登録
手順4の「承認済みのJavaScript生成元」に、今発行されたNetlifyのURLを追加します。（まだ追加していない場合）
6
アプリにログインする
▾
1
NetlifyのURLをブラウザで開く
MailFocusのログイン画面が表示されます。
2
Client IDを入力して「Googleアカウントでログイン」
Client ID
1234567890-abcd.apps.googleusercontent.com
G Googleアカウントでログイン
3
Googleの確認画面への対処
⚠️
「このアプリはGoogleで確認されていません」と表示された場合:

「詳細」をクリック → 「MailFocusに移動（安全でない）」をクリック

自分で作ったアプリなので安全です。
4
Gmailへのアクセスを「許可」する
「メールの読み取りと送信を許可する」という画面が出たら「許可」をクリック。これでMailFocusが起動します。
🎉
セットアップ完了！未読メールとスターメールが表示されます。
PWAとしてインストール
📱 iPhone / iPad
1. Safari でアプリを開く
2. 下部の共有ボタン（□↑）をタップ
3.「ホーム画面に追加」を選択
4. 「追加」をタップ
🤖 Android
1. Chrome でアプリを開く
2. メニュー（⋮）をタップ
3.「ホーム画面に追加」を選択
4. 「追加」をタップ
準備完了 🚀
