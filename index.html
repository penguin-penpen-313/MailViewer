Client ID取得手順

  1. Google Cloud Console を開く
  2. 新しいプロジェクトを作成
  3. APIとサービス → ライブラリ → Gmail API を有効化
  4. APIとサービス → OAuth同意画面 → 外部で作成
    「対象」メニューでテストユーザーに自分のアドレスを追加
  5. 認証情報 → OAuthクライアントID → ウェブアプリケーション
  6. 承認済みJS生成元に を追加
  7. Client IDをコピーしてログイン画面に貼り付け
